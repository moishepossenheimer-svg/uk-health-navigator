HOW TO UPLOAD TO GITHUB
1. Extract this ZIP.
2. Open your GitHub repository: uk-health-navigator.
3. Click Add file > Upload files.
4. Drag ALL files and folders from inside this extracted folder into GitHub.
5. Click Commit changes.
6. Wait 1-3 minutes.
7. Your GitHub Pages website should update automatically.
