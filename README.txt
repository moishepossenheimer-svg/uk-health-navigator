UK Health Navigator

Free hosting recommendation:
- GitHub Pages: free for public repositories.
- Cloudflare Pages: generous free plan.

GitHub Pages quick setup:
1. Create a GitHub account.
2. Create a public repository called uk-health-navigator.
3. Upload all files from this extracted folder.
4. Go to Settings > Pages.
5. Source: Deploy from branch.
6. Branch: main, folder: /root.
7. Your site will appear at https://YOURNAME.github.io/uk-health-navigator/

This site is for navigation and education only, not diagnosis or treatment.
